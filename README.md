# Typing Game

A simple browser-based typing practice game built with HTML, CSS, and JavaScript.

# Live link

->> https://suman-max10.github.io/TypingGame/

## Features

- Random passage selection on load and reset
- Live tracking for time, mistakes, WPM, and CPM
- Keyboard-first gameplay with click-to-focus support
- Reset button to start a new round

## How to Run

1. Open `index.html` in a web browser.
2. Start typing the highlighted text.
3. Use **Try Again !!** to load a new passage and restart the timer.

## Project Files

- `index.html` - page structure
- `style.css` - game styling
- `script.js` - game logic and score tracking

## Notes

- No build step or external dependencies are required.
- The game works as a static front-end project.
